# 🛍️ Bazar Sandra Presentes

Institutional website for **Bazar Sandra Presentes**, built with pure HTML and CSS as a portfolio project. The goal was to create a digital presence for a bazaar with over 34 years of tradition, showcasing its products and making it easier for customers to get in touch.

---

## 📋 About the Project

**Bazar Sandra Presentes** is a brick-and-mortar store located in Londrina – PR, Brazil, selling fashion, toys, stationery, gifts, and personalized mugs. This website was created to strengthen the business's online presence, providing information about product categories, contact via WhatsApp and Instagram, and a Google Maps location link.

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Tomas5014/bazar-web-site.git
   ```

2. Navigate to the project folder:
   ```bash
   cd bazar-web-site
   ```

3. Open `index.html` in your browser.

> No external dependencies or installations required.
